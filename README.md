## HowToUse

zaima是一个无聊的练手，用来在命令行中查看[摸鱼王Quin](https://www.douyu.com/3614)有没有在播。

对zaima执行``$ chmod a+x zaima.py``之后，将文件路径加入到系统环境变量中。

喜欢的话可以去掉后缀py，直接输入``$ zaima``执行。
